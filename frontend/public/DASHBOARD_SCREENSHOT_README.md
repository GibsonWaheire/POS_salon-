# Dashboard screenshot

The dashboard screenshot used on the **About Us** page lives in **`src/assets/dashboard-screenshot.png`** and is **imported** so it goes through the bundler (hashing, optimization).

**To update it:**

1. Run the app and open the Dashboard (`/dashboard`).
2. Enable **Demo Mode** for sample data.
3. Take a full-page or window screenshot.
4. Save it as `frontend/src/assets/dashboard-screenshot.png`, replacing the existing file.
